Authorization Validity Checker

This tool verifies claim authorization data from a submitted XML file against an insurer's exported authorization list (Excel Sheet 2).

  • Checks for valid Emirates ID format and member info.
  • Ensures Package Name matches insurer records.
  • Validates prior authorization codes against known formats.
  • Cross-checks activity codes and clinicians with the XLSX file.
  • Flags missing or mismatched information, VAT inconsistencies, and duplicate ICD codes.

Upload the XML and the Excel file, then click "Run Checker" to begin.
