Authorization Validity Checker

What this script does:

This tool allows you to upload a Claim Submission XML file along with an Excel export (Sheet 2) containing authorization records. It validates Emirates ID format, verifies the correctness of authorization codes, ensures clinician and activity details match the export, and checks for duplicate diagnoses, invalid packages, VAT inconsistencies, and other discrepancies. After processing, the tool will show any remarks that need attention.
