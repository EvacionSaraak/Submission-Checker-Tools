Drug Lookup System

What this tool does:

This tool allows you to look up drug information by uploading a Shafafiyah drugs XLSX file. Use the Drug Lookup tab to search for individual drug codes. Use the XML Analysis tab to validate drug codes within XML claims files. Please make sure to use the latest version of the XLSX from Shafafiyah.

Instructions:
1. Upload the drugs XLSX file (the second sheet will be used).
2. Select a mode: Drug Lookup or XML Analysis.
3. For Drug Lookup: enter a Drug Code or Package Name and click search.
4. For XML Analysis: upload an XML file and click analyze (feature pending).
5. To calculate pricing, select a displayed row and then input the desired quantity.
